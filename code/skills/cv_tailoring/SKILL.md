# Skill: CV Tailoring

## Purpose
Refine a user's CV for a specific opportunity.

## Inputs
- Original CV text
- Opportunity title
- Organization
- Description
- Requirements

## Steps
1. Read CV text.
2. Read opportunity requirements.
3. Identify relevant skills and experience.
4. Reorder and rewrite CV sections.
5. Remove irrelevant emphasis.
6. Keep all information truthful.
7. Output only the final CV content.

## Rules
- Do not invent fake experience.
- Do not add fake publications.
- Do not add fake degrees.
- Do not add placeholders in final CV.
- Do not include explanation text inside the CV.

## Tool Used
- Ollama

## Output
- Refined CV draft
