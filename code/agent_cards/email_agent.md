# Email Sender Agent

## Role
Sends application emails with CV and cover letter attachments.

## Inputs
- Recipient email
- Email subject
- Email body
- CV PDF
- Cover letter PDF

## Outputs
- Sent email
- CSV tracker update

## Tools
- Gmail SMTP
- Application Tracker
- Agent Logger

## Risk Level
High

## Approval
Requires explicit /sendemail command every time.
