# ApplyPilot Evaluation Plan

## Success Criteria
- CV upload works.
- Opportunity search works.
- CV tailoring works.
- Cover letter and email draft are generated.
- Evaluation Agent checks the package.
- Email is sent only after /sendemail.
- CV PDF and cover letter PDF are attached.
- Replies are tracked.
- Telegram notification is sent for replies.
- Logs and CSV tracker preserve evidence.

## Safety Criteria
- No fake CV content.
- No email without approval.
- No API keys in GitHub.
- No raw private CV data sent to cloud LLMs by default.
