# ApplyPilot Architecture

ApplyPilot is a Telegram-based agentic workflow.

Core agents:
1. Campaign Orchestrator Agent
2. Opportunity Discovery and Ranking Agent
3. CV Tailoring Agent
4. Application Drafting Agent
5. Evaluation and Communication Agent

Tools:
- Ollama
- Tavily
- Gmail SMTP/IMAP
- CSV tracker
- PDF exporter
