# Approval Policy

## Auto-Allowed Actions
- General chat
- CV upload and reading
- Opportunity search
- Opportunity ranking
- CV draft generation
- Cover letter draft generation
- Reply checking

## Approval Required Actions
- Sending emails
- Applying to any position
- Submitting forms
- Deleting files
- Sharing private user data externally

## Email Rule
ApplyPilot must only send email after this explicit command:

/sendemail opportunity_number recipient@email.com
